# 超声报告领域词典增强中文分词系统

## 1. 项目目标

本项目面向自然语言处理课程，研究超声诊断报告中的中文分词问题。医学报告包含大量器官、部位、病灶、回声、血流和测量术语，通用分词容易将这些术语拆碎，因此项目使用医学词典增强分词。

项目不进行疾病分类、标签预测、实体识别或临床诊断。

## 2. 数据

系统使用三类超声报告：

| 类型 | 数量 |
| --- | ---: |
| 甲状腺 | 2457 |
| 乳腺 | 3513 |
| 肝胆胰脾 | 1394 |
| 合计 | 7364 |

每条数据包含报告编号、报告文本、图像文件名、原始标签和数据划分。项目只读取报告文本、编号和数据划分，不使用原始数字标签。

## 3. 数据预处理

预处理脚本为：

```powershell
python scripts/preprocess_data.py
```

处理内容：

- 读取 UTF-8 BOM JSON
- 修复可能的乱码
- 删除多余空白
- 统一英文和中文分隔符
- 保留 `_2DS_`、`_3DS_`、`_SCM_` 等测量占位符
- 生成清洗数据、统计信息和 Jieba 用户词典

## 4. 医学词典

词典由四部分组成：

1. 人工分类词典：器官、部位、病灶、回声、边界、形态、血流、状态、测量。
2. 原始技术词表：`USData/key_technical_words.txt`。
3. 数据集高频医学短语：根据频次和医学后缀规则自动扩展。
4. 强制保护短语：`resources/medical_phrases.txt`。

停用词独立保存在：

```text
resources/stopwords.txt
```

## 5. 分词算法

### 5.1 Jieba + 医学词典

系统把完整医学词典加载到 Jieba，并在分词前保护复杂短语，例如：

- 囊实混合回声结节
- 以囊性为主
- 右叶近峡部
- 周边及内部可探及血流信号
- 甲状腺全切术后

处理流程：

```text
清洗文本 → 短语保护 → Jieba 分词 → 停用词过滤
```

### 5.2 正向最大匹配 FMM

从文本左侧开始，在医学词典中寻找最长匹配词。若没有匹配词，则保留当前单字。

### 5.3 逆向最大匹配 RMM

从文本右侧开始寻找最长匹配词。它可以与 FMM 对比，观察中文切分方向带来的差异。

### 5.4 词性标注

系统使用 `jieba.posseg` 给分词结果补充词性，并标记词语是否存在于医学词典中。

## 6. 算法对比

系统将三种算法的词语映射回原文本位置，比较词边界：

- 两种算法都产生的边界
- 只有部分算法产生的边界
- 两两词边界一致率
- 分歧位置附近的原文上下文

该功能用于解释不同分词算法的行为差异。

## 7. 人工金标准评测

人工金标准位于：

```text
resources/segmentation_gold.json
```

共包含 30 条报告，甲状腺、乳腺和肝胆胰脾各 10 条，覆盖：

- 正常报告
- 单发和多发结节
- 囊实混合病灶
- 复杂部位
- 术后描述
- 淋巴结
- 血流信号
- 测量占位符

运行：

```powershell
python scripts/evaluate_segmentation.py
```

评测结果写入：

```text
processed_data/segmentation_metrics.json
```

指标按词边界计算：

```text
Precision = 正确预测边界数 / 预测边界总数
Recall = 正确预测边界数 / 人工边界总数
F1 = 2 × Precision × Recall / (Precision + Recall)
```

当前结果：

| 算法 | Precision | Recall | F1 |
| --- | ---: | ---: | ---: |
| Jieba + 医学词典 | 0.8246 | 0.9691 | 0.8910 |
| FMM | 0.7311 | 0.8969 | 0.8056 |
| RMM | 0.7607 | 0.9175 | 0.8318 |

## 8. 网页系统

网页提供三个视图：

- 报告分词：三种算法、词性、未登录词、占位符和边界差异
- 分词评测：总体指标、分器官指标和错误案例
- 词典构建：词典规模、分类、停用词、保护短语和动态扩展词

后端仅保留四个接口：

| 接口 | 功能 |
| --- | --- |
| `GET /api/report/sample` | 获取报告样本 |
| `POST /api/analyze` | 执行三种分词和对比 |
| `GET /api/stats` | 获取数据与评测统计 |
| `GET /api/dictionary` | 获取词典资源 |

## 9. 局限与后续优化

- 分词质量仍受医学词典覆盖率影响。
- 人工金标准只有 30 条，后续应扩大标注规模。
- 动态高频词可能包含局部噪声，需要人工审核。
- 停用词过滤会改变原始切分展示，应同时保留清洗前文本作为参照。
- 可加入用户纠错功能，将错误切分词加入保护短语。

## 10. 总结

本项目完成了一个聚焦超声报告中文分词的课程系统。系统通过医学词典、短语保护和停用词过滤增强 Jieba，并与正向、逆向最大匹配进行对比，再使用人工金标准评价分词质量。项目主题明确、结果可解释，适合作为中文分词课程实验。
