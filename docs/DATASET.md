# 超声诊断报告数据集介绍

## 数据集概况

本项目使用的是超声诊断报告文本数据集，主要包含三类检查部位：

- 甲状腺超声报告
- 乳腺超声报告
- 肝胆胰脾超声报告

数据集总计包含 **7364 条超声报告文本**，适合用于医学自然语言处理课程任务，例如分词、词典构建、关键词抽取、正则匹配、实体识别、文本规范化和模板报告生成。

## 原始数据文件

原始数据位于 `USData/` 目录下：

```text
USData/
  new_Thyroid2.json       # 甲状腺超声报告
  new_Mammary2.json       # 乳腺超声报告
  new_Liver2.json         # 肝胆胰脾超声报告
  key_technical_words.txt # 医学技术词表
```

## 数据字段说明

每条报告数据包含以下字段：

| 字段 | 含义 |
| --- | --- |
| `uid` | 报告或病例编号 |
| `finding` | 超声所见文本，是本项目 NLP 处理的核心字段 |
| `image_path` | 对应超声图像文件名 |
| `labels` | 报告类别标签 |
| `split` | 数据划分，包括 `train`、`test`、`val` |

示例报告文本：

```text
甲状腺大小形态如常，腺体回声均匀，未见明确占位性病变，
CDFI示腺体内未见异常血流信号。双侧颈部未见明显肿大淋巴结。
```

## 占位符说明

报告文本中存在一些标准化占位符，用于表示尺寸、方位、比例等医学信息：

| 占位符 | 含义 |
| --- | --- |
| `_2DS_` | 二维尺寸 |
| `_3DS_` | 三维尺寸 |
| `_SCM_` | 厘米数值 |
| `_SMM_` | 毫米数值 |
| `_Loc_` | 钟点或方位 |
| `_LocR_` | 范围方位 |
| `_r_` | 纵横比 |

各类报告中的占位符数量如下：

| 数据类型 | 主要占位符统计 |
| --- | --- |
| 甲状腺 | `_2DS_`: 2662，`_3DS_`: 720，`_SCM_`: 180，`_r_`: 63，`_SMM_`: 1 |
| 乳腺 | `_2DS_`: 2648，`_SCM_`: 2437，`_Loc_`: 1936，`_3DS_`: 332，`_LocR_`: 94，`_SMM_`: 64 |
| 肝胆胰脾 | `_SCM_`: 590，`_2DS_`: 458，`_3DS_`: 9，`_SMM_`: 1 |

## 文本内容特点

该数据集具有典型的医学报告文本特点：

- 专业术语密集，例如 `低回声结节`、`边界清晰`、`形态规整`、`血流信号`、`淋巴结`
- 句式较固定，适合规则抽取和模板生成
- 存在大量否定表达，例如 `未见`、`未探及`、`无明显`
- 包含部位、尺寸、回声、边界、形态、血流等结构化信息
- 不同检查部位具有不同的高频词和描述模式

## 预处理数据

为了让后续 NLP 处理更稳定，本项目新增了独立的数据预处理脚本：

```text
scripts/preprocess_data.py
```

运行方式：

```powershell
python scripts/preprocess_data.py
```

预处理后会生成 `processed_data/` 目录：

```text
processed_data/
  thyroid_clean.json
  mammary_clean.json
  liver_clean.json
  dataset_stats.json
  medical_dictionary.json
  preprocess_summary.json
```

预处理内容包括：

- 兼容 UTF-8 BOM
- 修复部分文本编码异常
- 去除多余空白字符
- 保留并统计医学占位符
- 为每条报告生成分词预览
- 生成数据统计文件
- 生成医学词典文件

预处理统计：

| 数据类型 | 总数 | 清洗后有变化的报告数 | 平均文本长度 |
| --- | ---: | ---: | ---: |
| 甲状腺 | 2457 | 164 | 115.28 |
| 乳腺 | 3513 | 23 | 100.06 |
| 肝胆胰脾 | 1394 | 1 | 161.95 |

## 在本项目中的用途

本数据集在项目中主要用于以下 NLP 功能：

- 医学报告分词
- 医学词典构建
- 高频词和 TF-IDF 关键词抽取
- 基于正则表达式的尺寸、部位、血流、边界、形态抽取
- 医学实体识别
- 文本规范化
- 模板化报告生成
- 报告质量检查
- 相似报告检索

## 说明

本项目仅用于自然语言处理课程学习和演示，不用于真实临床诊断。报告生成结果是基于规则和模板的文本摘要，不代表医学诊断意见。
